平均响应时间算法

经过反复测试，临界值为平均响应时间乘以2万次（如200ms为400万次），不会再变化。

如果当天请求超过平均响应时间乘以2万次，则不应该再计算新值。

最终算法

// count,restime
let avg = count > 10E5 ? val : ((restime + parseFloat(val) * (count - 1))/count);
